# 数学分析二复习讲义

## 📖 项目介绍
这是一份个人整理的数学分析（第二册）期末复习讲义，备考期间一边梳理知识框架一边写成，初衷是给自己理清复习主线，也希望能帮到同样在学数分的同学。

讲义不是教材的复刻，重点梳理了期末核心考点、通用解题方法、高频易错点，每道例题都附带思路引导，适合期末冲刺复习，也适合新课学习时当作知识点提纲。

> 注：讲义仅为个人学习整理，难免有疏漏错误，仅供参考，一切以教材和课堂内容为准。

## 📁 内容涵盖
- 数项级数（敛散性判别、典型题型、易错点）
- 函数项级数与一致收敛
- 幂级数与傅里叶级数
- 多元函数微分学
- 重积分
- 曲线积分与曲面积分

## 📥 使用方式
1. **直接使用**：下载根目录下的 `数分二复习讲义.pdf` 即可阅读
